# **App Name**: School Navigator

## Core Features:

- Menu Navigation: Display a menu icon for easy navigation.
- School Logo: Show the school's logo clearly.
- School Name Display: Clearly display the school's name.
- Profile Photo Display: Display the profile photo of a key school representative.

## Style Guidelines:

- Primary color: Light teal (#94D6D2) to reflect serenity.
- Background color: Very pale teal (#F0F8F7), same hue as the primary but desaturated for subtlety.
- Accent color: Soft green (#87D69A), analogous to teal, but shifted in brightness.
- Font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth; to be used in the headers and in short sections of text.
- Top bar layout similar to the provided image, responsive for different screen sizes.
- Use clear and simple icons for navigation.
- Subtle transitions and animations when navigating the school website.
